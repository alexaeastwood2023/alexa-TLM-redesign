Course Offering

"Changing Life Through Tech"

Disrupting the system of mass incarceration through in-prison technology education, providing meaningful employment and reentry support after release.

Courses Available:

  • Intro to Tech Pathways (youth): this 3-month course for youth students covers concepts in the tech industry and provides an introduction to HTML and CSS
  • Web Development Fundamentals: this 6-month course for adult students covers basic computer skills and front-end web development technologies like HTML, CSS, and JavaScript
  • MERN Development: this 6-month course for adult students covers back-end and full-stack web development technologies like Node, Express, React, and MongoDB
  • Intro to Web Development (youth): this 3-month course for youth students covers intermediate front-end web development skills such as JavaScript
  • Front-end Development Fundamentals (youth): this 3-month course for youth students covers advanced front-end web development skills such as React
